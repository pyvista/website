Projects

PyVista is an ecosystem with many interrelated projects and repositories. The following are a list of projects the PyVista developers support:

PyVista

3D plotting and mesh analysis through a streamlined interface for the Visualization Toolkit (VTK)

  • pyvista/pyvista
  • http://docs.pyvista.org
  • Stable

PyMeshFix

Python Wrapper for MeshFix: easily repair holes in PyVista surface meshes

  • pyvista/pymeshfix
  • http://pymeshfix.pyvista.org
  • Ready, but changing

TetGen

A Python interface to the C++ TetGen library to generate tetrahedral meshes of any 3D polyhedral domains

  • pyvista/tetgen
  • https://tetgen.pyvista.org
  • Stable

PyACVD

Python implementation of surface mesh resampling algorithm ACVD

  • pyvista/pyacvd
  • No documentation yet!
  • In the works...
